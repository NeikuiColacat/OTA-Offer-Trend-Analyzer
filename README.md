# OTA-Offer Trend Analyzer

## 🎯 项目简介

本项目旨在通过自动化爬虫技术，实时获取字节跳动、阿里巴巴、腾讯三家公司的头部AI人才计划招聘岗位详情，并提数据清洗标准化。无论是求职者洞察行业动向，还是数据分析师研究招聘趋势，这个工具都能提供结构化、可靠的数据支持。

## ✨ 核心功能

### 1. **多源爬虫集成**

- 🎯 **字节跳动** (`jobs.bytedance.com`) - 监听 API 响应，精准抓取岗位详情
- 🎯 **阿里星计划** (`talent.alibaba.com`) - 支持动态翻页，累积聚合多页数据
- 🎯 **腾讯青云计划** (`join.qq.com`) - 利用详情 API，获取结构化岗位信息

### 2. **智能数据清洗**

三套定制化清洗函数，按各公司数据特性优化：

- **字节跳动**：剥离冗余嵌套结构、统一城市字段、时间戳 ISO 化
- **阿里星**：删除追踪字段、提取关键维度（批次/组织/毕业时间范围）
- **腾讯**：压缩部门嵌套信息、去重、规范地址列表

### 3. **统一输出格式**

每条记录包含：

```json
{
  "source": "bytedance/alibaba_star/tencent",
  "job_id": "岗位唯一标识",
  "title": "岗位名称",
  "category": "岗位类别",
  "locations": ["城市1", "城市2"],
  "description": "岗位描述",
  "requirement": "岗位要求",
  "publish_time_iso": "2026-01-27T10:30:00+00:00",
  ...
}
```

## 📁 项目结构


## 🚀 快速开始

### 环境要求

```bash
uv pip install -r requirements.txt

uv run playwright install chromium
```

### 运行爬虫

```bash
python main.py
```

## 💡 使用场景

1. **求职者** - 了解目标公司最新招聘方向与岗位分布
2. **数据分析师** - 分析招聘规模变化、职位热度排行
3. **HR** - 对标竞品招聘策略、人才需求动向
4. **研究人员** - 科技行业人才流动、技术栈演进研究
